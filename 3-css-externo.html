Archivo externo para estilos CSS

A medida que vamos necesitando crear más y más reglas de css, vemos que empieza a se impráctico tener tantas reglas de css junto con todo el código html, en un mismo cuerpo enorme de código que se hace dificil de manejar.

La mejor opción en este caso (y en general) es extraer todas esas reglas de estilo (CSS) a un archivo específico e independiente, destinado únicamente a los estilos CSS.

Y para que el navegador web sepa a dónde debe buscar los estilos cuando esté cargando nuestra página, tenemos que agregar una instrucción especial <link> dentro del <head> de la página, con una propiedad href para indicar la ubicación del archivo de estilos (similar a la propiedad href que usamos en los anchors) de esta forma: 

        
      <html>
      <head>
        <link rel="stylesheet" type="text/css" href="estilos.css" />
      </head>
      <body>
      ...

De esta forma, ya no tendríamos que estar "insertando" nuestras reglas CSS dentro del mismo archivo con el código html, sino que mantenemos todo lo relacionado con los estilos CSS en su propio archivo (en este esmplo, sería ese estilos.css)

👎 Una posible desventaja de esto, al menos mientras estamos en fase de desarrollo y ajustando los estilos de una página, es que como los estilos están en un archivo aparte, ya no será tán inmediato poder ver los cambios en directo en nuestro editor de códigos (sino que nos toca ir recargando la previsualización cada que cambiemos algo en el archivo de estilos)

👍 Aún así, la gran ventaja es poder tener organizado, en un solo sitio, todos los estilos que podemos compartir entre varias páginas de nuestro sitio (siempre que realmente se trate de estilos que sí son comunes a esas páginas)

También es posible usar una "plantilla" o archivo de estilos con todos esos estilos que compartimos (y reusamos) entre varias de (o todas) nuestras páginas, pero, en alguna página en específico insertar además alguna que otra clase adicional, que sea de uso exclusivo en esa página únicamente.

En ese caso símplemente agregamos primero la referencia a los estilos "comunes" y debajo el bloque de estilos específicos de esa página, así: 

        
      <html>
      <head>
        <link rel="stylesheet" type="text/css" href="estilos.css" />
        <style>
          ul {
            list-style-type: none;
          }
          li {
            padding: 2px 5px;
          }
        <style>
      </head>
      <body>
      ...
