Selectores con Clases CSS

Los selectores por tipo son muy útiles cuando queremos aplicar un mismo estilo a "todos" los elementos de ese tipo (o, como vimos anteriormente, a "todos los elementos de tipo X que se encuentren dentro de "Y") Sin embargo, muchas veces necesitamos de algún atributo o "característica especial" adicional por la cual poder seleccionar sólo a los elementos que nos interesan.
