Selectores CSS

Los selectores definen sobre qué elementos (específicos o en forma generalizada) se aplicarán las propiedades que definimos en una regla de CSS.

Se rigen por unas especificaciones (o reglas) escritas en base al tipo de elemento que queremos seleccionar (para aplicarle cierto estilo) o algunas de sus atributos, clases, identificadores, a su posición relativa o específica dentro del DOM html, entre otros.

Como el ejemplo más sencillo, veamos este bloque de código html: 

        
    <div>
      <p>
        Primer párrafo dentro de un div
      </p>
    <div/>
    <div>
      <div>
        <p>
          Segundo párrafo dentro de un div
          "anidado" dentro de otro div
        </p>
        <a href="#">Click aquí</a>
      <div/>
    <div/>
    <p>
      Un párrafo "libre de divs"
    </p>

Algunos selectores útiles que podríamos formar (por ahora sólo con selectores de "tipo") para aplicar distintos estilos en cada caso serían, por ejemplo:

para seleccionar todos los elementos de tipo div usamos simplemente:
div
para seleccionar todos los elementos de tipo párrafo y al mismo tiempo los anchors, separamos por coma:
p, a
para seleccionar todos los párrafos que estén dentro de algún div (lo cual excluye al último párrafo) :
div p
Sólo divs que estén dentro de un div:
div div
para seleccionar todos los elementos dentro de algún div (sean otros divs, párrafos o cualquier otro tipo de elementos):
div *
👉 No solo usaremos selectores para poder aplicarle estilos a los elementos de nuestro DOM html. También se usan estos selectores para programar acciones que queremos que se ejecuten sobre algunos de estos elementos en particular, mediante scripts (en lenguaje javascript). De ahí la importancia de entender y dominar el uso de los selectores adecuados
