HTML y CSS juntos

Podemos incluir los estilos (css) de nuestro HTML dentro del mismo archivo .html

Para eso, insertamos un tag <style> dentro del primer bloque o conjunto de elementos que tiene una página web: el <head> (en la primera parte del html) de esta forma: 

        
    <head>
      <style>
        body {
          margin: auto;
          padding: 10px;
          max-width: 800px;
        }
        p {
          font-size: large;
          margin-top: 20px;
        }
      <style>
    <head>

Luego ponemos todo nuestro html (el que contiene los elementos que queremos mostrar) en el propio "cuerpo" de la página, en el bloque <body> así: 

        
    <body>
      <div>
        <p>Este párrafo está dentro de un
          contendor <div>, en el cuerpo
          principal de la página (body)
        </p>
        <img src="images/foto.jpeg">
      <div>
    <body>

Todo esto estaría contenido dentro de un mismo archivo (por ejemplo: index.html) con la ventaja de que, al abrir este archivo .html en un editor de código como vs-code, es posible ir viendo una previsualización de nuestros cambios "en directo" a medida que vamos cambiando tanto el código html como las reglas de CSS
