Términos Básicos

HTML

El Lenguaje de Marcado de Hipertexto (en inglés: HyperText Markup Language HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

A pesar de que su nombre podría confundir, el HTML NO es un lenguaje de programación; es un lenguaje de marcado que define la estructura de un contenido (típicamente una página de un sitio web), y consiste en una serie de elementos que usamos para encerrar diferentes partes de ese contenido que queremos presentar, para que se vean y dispongan en la página de una manera específica.

Hipertexto

Hace referencia a los enlaces que conectan páginas web entre sí, ya sea dentro de un único sitio web o entre sitios web

Tags y Elementos

Un elemento HTML se distingue de otro texto en un documento mediante "etiquetas" (tags), que consisten en el nombre del tipo de elemento rodeado por "<" y ">".

Cada tag que agregamos al código html de nuestra página: una imagen, un div, un anchor (link), los párrafos, botones, tablas, etc.. cada uno de esos tags representan un único elemento de nuestra página.

Una forma rápida de entenderlo mejor, es decir que el tag es eso que empieza con un < y termina con >. Mientras que el elemento es todo lo que está contenido dentro del tag (tanto en sus atributos, como todo lo que esté encerrado entre el tag que abre y su cierre)

A pesar de que podemos tener (digaamos) unos 500 tags <div> en una misma página, cada uno de ellos representa un elemento único (aún y cuando luzca exactamente igual que los que le rodean) puesto que tiene información (o posición) propia dentro de esa estructura del documento.

Atributos

Algunas veces no basta con sólo escribir un tag para definir en detalle lo que necesitamos representar en un elemento, así que tenemos que agregar información adicional (además del contenido en sí). Los atributos proporcionan información adicional sobre los elementos HTML.

Por ejemplo, para crear un enlace (anchor o link) no sólo usamos la etiqueta <a> sino que además tenemos que especificar el "atributo" que le indica al navegador hacia qué dirección debe apuntar ese anchor (el atributo href)

Otro atributo muy común (y similar al href del anchor) sería el src que agregamos a los tags de imágenes (<img>) en este caso para indicar la ubicación de la imagen que queremos mostrar.

DOM

Una página HTML está formada por múltiples etiquetas HTML, anidadas una dentro de otra, formando una especie de árbol de etiquetas relacionadas entre sí. Así que este árbol se puede representar como un modelo de objetos del documento (que llamamos DOM por: Document Object Model) que podríamos "recorrer" mediante secuencias de comandos ("scripts") para poder manipular algunos de sus elementos (o activar acciones que queremos ejecutar en nuestra página).

En otras palabras, el DOM es una representación "ordenada" de toda la estructura de anidamiento de nuestro html (que podríamos graficar tal cual como un árbol)
