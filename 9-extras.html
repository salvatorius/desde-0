Extras sobre elementos usados en esta guía

Literal

A veces necesitamos representar algo como un tag directamente en nuestra página web (tal como los que están en esta página), por ejemplo, si queremos mostrar un código de ejemplo.

Si lo intentamos escribir así directamente en nuestro código html, el intérprete (o browser) intentará ejecutar ese código (en vez de mostrarlo directamente)

Como ejercicio, puedes intentar escribir un tag cualquiera en tu editor de código e intentar que se vea así tal cual en el resultado final de la página

Una de las formas que tenemos para lograr eso, es reemplazando ciertos caracteres especiales (como el <) por una expresión equivalente que resulta ser una representación "literal" de ese mismo caracter. En el caso del < sería: &lt; el literal que representa el caracter de "menor que" (o, en inglés, "less-than")

Algunos otros literales útiles son:

Caracter Literal En inglés
& &amp; ampersand
< &lt; less than
> &gt; greater than
" &quot; quotes
' &apos; apostrophe
→ &rarr; right arrow
← &larr; left arrow
˜ &tilde;

De esta tabla, por ahora, sólo vamos a estar usando los 2 primeros

Formato especial de Código

Cuando queremos representar un fragmento de código (textual) en nuestra página, hacemos uso del tag <code> que los navegadores interpretan (y representan) adecuadamente como código (con un tipo de letra monoespaciada, un fondo o color ligeramente distintos, etc)

Si escribimos este código en nuestro editor:

        
    <p>
      Así representamos una imagen: 
      <code>&lt;img src="nuevafoto.png"/><code>
    </p>

Veremos algo como esto en el navegador:

Así representamos una imagen: <img src="nuevafoto.png"/>

Nota: Se puede ver que: para poder representar el tag <img> (y que se muestre así tal cual en nuestra página) tuvimos que reemplazar el < por el literal &lt;

Bloques de "pre" formato

Ya vimos cómo representar código (con un formato monoespaciado) pero también podemos crear un bloque completo de código (tal como ese que usamos arriba para mostrar el ejemplo de cómo usar el <code>)

Eso se logra encerrando el contenido en un tipo particular de contenedor <div> (más adelante veremos muchas otras variantes de los contenedores o "divs") En este caso, el tag que usaremos es el <pre>, el cual podemos también usar en conjunto con el tag <code> para crear un bloque "pre-formateado" de código: 

        
    <pre>
      <code>
        &lt;p>
          Este es un bloque completo de código de ejemplo
          incluyendo tags html
        &lt;/p>
        &lt;img src="ejemplo.png"/>
      <code>
    <pre>
Será renderizado (y visible en nuestro navegador) así: 
          
    <p>
      Este es un bloque completo de código de ejemplo
      incluyendo tags html
    </p>
    <img src="ejemplo.png"/>
Nuevamente recordamos reemplazar el < por el literal &lt; para que se pueda mostrar correctamente el tag con su respectivo <
